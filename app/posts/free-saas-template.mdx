This is a free SaaS template using Nextjs and Supabase. It provides a fully functional authentication
implementation powered by Supabase. It also includes a blog feature using MDX for content management.
The template is fully responsive and includes a dark mode toggle.

## Features

<LogoCloud />

- Next.js
- Database, Auth & Backend via Supabase (full auth implemented)
  - PKCE flows (forgot password, email verification)
  - supabase/ssr
- Subscriptions via Stripe
- Blog powered by Mdx. Use any component and Github-flavoured markdown
- Dark mode toggle and theming support by [next-themes](https://github.com/pacocoursey/next-themes)
- Tailwind 3 and [shadcn/UI](https://ui.shadcn.com/) included for easy styling and reusable components
